## Description
<p>Rock-Con is a web application for:</p>
<p>:guitar: Beginner singers and groups :guitar:</p>
<p>:moneybag: Organizations looking to reach new audiences :moneybag:</p>
<p>:musical_note: People who want to listen lovely musiciants :musical_note:</p>
<p>So music groups or artists can add info about them and connect with organizations. Organizations also can sign up, sign in, correct info about themselves and add info about concerts by choosing member, time and date. </p>
